# ReservaPo - Proyecto FullStack

Aplicación FullStack para gestión de reservas de negocios y profesionales.

## Tecnologías utilizadas

### Frontend
- React
- Vite
- React Router DOM
- Context API
- CSS

### Backend
- Node.js
- Express
- PostgreSQL
- JWT
- Bcrypt
- Zod
- Jest

# Instalación

## 1. Clonar repositorio

```bash
git clone URL_DEL_REPOSITORIO
```

---

## 2. Instalar frontend

```bash
cd frontend
npm install
```

---

## 3. Instalar backend

```bash
cd ../backend
npm install
```

---

# Variables de entorno Backend

Crear archivo `.env` dentro de `/backend`

```env
DB_USER=
DB_HOST=
DB_NAME=
DB_PASSWORD=
DB_PORT=
SECRET_KEY=
```

---

# Ejecutar frontend

```bash
cd frontend
npm run dev
```

---

# Ejecutar backend

```bash
cd backend
npm run dev
```

---

# Ejecutar tests

```bash
cd backend
npm test
```
